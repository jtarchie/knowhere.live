Knowhere

Knowhere provides the querying of Open Street Map data, with geospatial scripting (via Javascript), to display it on a map (via GeoJSON).

Keyboard Shortcuts

  • ctrl / cmd+shift+d - load up the editor
  • ctrl / cmd+shift+f - execute script in the editor
